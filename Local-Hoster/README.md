# Local-Hoster

Portable, pretty local dev runner with auto port-scan and a calm UI.

## Install & Run (pick one)

- One-off (no install):
```bash
npx local-hoster@latest
bunx local-hoster@latest
```

- Global install (adds `zz`):
```bash
yarn global add local-hoster   # or
npm i -g local-hoster          # or
pnpm add -g local-hoster       # or
bun install -g local-hoster
zz
```

- Curl:
```bash
curl -fsSL https://raw.githubusercontent.com/REPO_OWNER/Local-Hoster/HEAD/local-hoster.sh -o zz
chmod +x zz
./zz
```
